About the Project

This project implements a binary text classifier to distinguish between Sports and Politics news articles. Using a dataset of over 60,000 articles, we trained and evaluated three different machine learning models to determine the most effective approach for this natural language processing task.

Model Performance Comparison

We compared three models: Naive Bayes (using Bag of Words), Linear SVM (using TF-IDF), and Logistic Regression (using TF-IDF). All models performed exceptionally well, achieving over 97% accuracy.

Model Feature Representation Accuracy
Multinomial Naive Bayes Bag of Words (CountVectorizer) 97.45%
Linear SVM TF-IDF 97.55%
Logistic Regression TF-IDF 97.61%

Methodology

1. Data Collection

We utilized the AG News Dataset, extracting Class 1 (World/Politics) and Class 2 (Sports). The dataset was balanced with 30,000 training samples and 1,900 test samples per class.

2. Preprocessing

Text data was cleaned, and we concatenated the "Title" and "Description" fields to provide richer context for classification.

3. Training

We used scikit-learn pipelines to streamline the vectorization and training process. 5-fold cross-validation was used during development to ensure robustness.

Conclusion

The project demonstrates that for distinct topics like Sports and Politics, linear models with simple feature representations (TF-IDF) are highly effective. Logistic Regression proved to be the best performing model, offering a perfect balance of accuracy and interpretability.
